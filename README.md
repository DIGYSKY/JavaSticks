# Baton

## Description

Baton is a simple game of logic that consists in removing sticks from a table. The player who removes the last stick loses.

## Usage

**For first time use**

```bash
./baton --compile-run
```
**For next uses**

```bash
./baton --run
```

**For more information**

```bash
./baton --help
```

## Requirements

- Java 21 or higher
